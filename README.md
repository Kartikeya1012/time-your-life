# Time Your Life

Time Your Life is a basic timetable scheduler where users can create their own timetables. The web application allows users to easily add and remove events and customize their timetables to fit their schedules.

## Features

- User-friendly interface
- Easy event creation and deletion
- Customizable timetable view
- Intuitive drag-and-drop functionality

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express
- EJS

## How to Use

1. Clone this repository.
2. Install dependencies using `npm install`.
3. Start the server using `node app.js`.
4. Navigate to `http://localhost:3000` in your browser.
5. Use the "Add Event" button to add new events to your timetable.
6. Drag and drop events to rearrange your timetable.
7. Use the "Remove" button to delete events.
